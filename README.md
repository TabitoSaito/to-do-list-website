# to do list website
Small website to manage todo-lists

## Features
- Create multiple todo-lists for each user
- Create multiple todos in lists
- Change status of todos to TODO | IN PROGRESS | COMPLETED

## Tech Stack
Python, Flask, SQLAlchemy, SQLite, Bootstrap5, WTF-Forms

## Installation
```bash
pip install -r requirements.txt
python main.py
```
website can be reached with [http://127.0.0.1:5000/]()
